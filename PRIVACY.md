# Privacy Policy

**Vision Guard PC**

Effective date: July 5, 2026

[Português](PRIVACY.pt.md) · **English**

---

## 1. Overview

Vision Guard PC is a desktop wellness application designed to encourage healthier computer-use habits through screen-time tracking, inactivity detection, reminders, breaks, and configurable usage limits.

Vision Guard PC follows a local-first approach. The current version does not require an online account and does not send the user's activity history to an external server.

---

## 2. Information stored by the application

Vision Guard PC may store the following information locally on the user's computer:

* active computer-use duration;
* inactive duration;
* completed, skipped, or interrupted breaks;
* daily usage history;
* selected language;
* wellness settings;
* daily usage limits;
* nighttime lock schedules;
* Focus Mode and Entertainment Mode state;
* window size, position, and maximized state;
* application runtime information required to prevent multiple instances.

This information is used only to provide the application's features.

---

## 3. Information Vision Guard PC does not monitor

Vision Guard PC does not monitor or record:

* text typed by the user;
* keyboard contents;
* passwords;
* private messages;
* visited websites;
* browser history;
* document contents;
* screenshots;
* camera input;
* microphone input;
* files opened by the user;
* precise geographic location.

The application only detects the amount of time since the last keyboard or mouse interaction. It does not record which keys were pressed or what actions were performed.

---

## 4. Data transmission

The current version of Vision Guard PC does not transmit usage history, application settings, or personal activity data to servers controlled by the developer.

Vision Guard PC does not currently include:

* advertising;
* analytics services;
* tracking pixels;
* cloud synchronisation;
* user accounts;
* behavioural profiling;
* third-party telemetry.

An internet connection may be required to download the application or receive updates through platforms such as GitHub Releases or the Microsoft Store. Those platforms operate under their own privacy policies.

---

## 5. Local storage

Application information is stored locally on the user's device.

On Windows, the typical storage location is:

```text
%LOCALAPPDATA%\VisionGuardPC
```

This directory may contain:

* the local SQLite database;
* application preferences;
* language settings;
* saved window state;
* temporary runtime information.

The exact files may change between application versions.

---

## 6. Data retention

Local information remains stored until one of the following occurs:

* the user deletes the application data manually;
* the application provides and the user uses a data-reset option;
* the application is uninstalled and its local data is removed;
* the operating system removes the application data.

Uninstalling the executable may not automatically remove all local settings or usage history. Users may delete the Vision Guard PC application-data directory to remove the remaining local information.

---

## 7. Data sharing

The developer does not sell, rent, trade, or share Vision Guard PC usage information with advertisers or data brokers.

Because the current version does not transmit local usage history to the developer, the developer normally has no access to that information.

---

## 8. Security

Vision Guard PC uses local storage and operating-system protections to reduce unnecessary exposure of user information.

However, no software or storage method can guarantee absolute security. Users should keep their operating system updated and protect their Windows account against unauthorised access.

---

## 9. Children’s privacy

Vision Guard PC is not specifically designed to collect information from children.

The current version does not require registration and does not intentionally collect personal information from children or adults.

Parents or guardians should supervise computer-use settings when the application is used by a minor.

---

## 10. Diagnostics and support

When requesting technical support, users may voluntarily provide information such as:

* the application version;
* operating-system version;
* error messages;
* screenshots;
* diagnostic logs.

Users should review support material before sharing it and remove any personal or sensitive information.

Information voluntarily submitted for support will be used only to investigate and respond to the reported problem.
